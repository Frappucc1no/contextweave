# Profiles

Profiles are guidance layers built on top of the shared ContextWeave protocol.

They do not replace the core protocol.

## Current Profiles

- `research-writing.md`
- `product-doc-collaboration.md`
- `software-project-coordination.md`

## How to Choose

Choose the profile that best matches:

- artifact type
- working style
- continuity risks
- evidence needs

## Shared vs Profile-Specific

Shared:

- file model
- write discipline
- stale retirement
- cold-start expectations

Profile-specific:

- evidence patterns
- artifact emphasis
- likely drift risks
- preferred review lens
